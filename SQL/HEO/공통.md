### 1. 구문차이

#### 페이징처리
  - MySQL
    > LIMIT N
  - Oracle
    > ROWNUM <= N

#### 현재날짜
  - MySQL
    > NOW()
  - Oracle
    > SYSDATE

#### 조건문
  - MySQL
    > IF(조건식, '참값', '거짓값')
  - Oracle
    > DECODE(열명, '값', '참값', '거짓값')

#### 날짜형식
  - MySQL
    > DATE_FORMAT(날짜열,'형식')
  - Oracle
    > TO_CHAR(날짜열, '형식')

#### 문자 추출
  - MySQL
    > SUBSTRING('ABCD', 2, 3)
  - Oracle
    > SUBSTR('ABCD', 2, 3)

#### 문자 결합
  - MySQL
    > CONCAT('AB', 'CD')
  - Oracle
    > CONCAT('AB', 'CD')
    > 'AB'||'CD'
