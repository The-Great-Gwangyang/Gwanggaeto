#File I/O
> 절대 경로와 상대 경로
* 절대경로
  - 절대 경로는 처음부터 시작하여 목적지까지의 절대적인 경로 
  - 최상위 /를 포함 
* 상대경로
  - 상대 경로는 현재 위치를 기준으로 하여 목적지까지의 상대적인 경로
  - 최상위 /를 거치지 않고도 이동이 가능
