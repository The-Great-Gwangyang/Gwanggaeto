# API(Application Programming Interface, 응용 프로그램 프로그래밍 인터페이스)
## API란?
 - 응용 프로그램에서 사용할 수 있도록, 운영체제나 프로그래밍 언어가 제공하는 기능을 제어할 수 있게 만든 인터페이스
 - 사용자가 다루기 위한 수단, 방법
 - jdk : Java 기본 내장 API
